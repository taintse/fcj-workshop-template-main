## Overview

This worklog documents my **13-week First Cloud Journey (FCJ) training program**, covering comprehensive AWS cloud development, modern DevOps practices, and full-stack application development. The program is structured into three main phases:

- **Phase 1 (Weeks 1-7):** AWS Fundamentals & Cloud Architecture
- **Phase 2 (Weeks 8-11):** Frontend Development & Cloud Integration
- **Phase 3 (Weeks 12-13):** Testing, Documentation & Project Finalization

---

## Weekly Breakdown

### Phase 1: AWS Fundamentals (Weeks 1-7)

**Week 1:** [Getting familiar with AWS and basic AWS services](1.1-week1/)
- Introduction to cloud computing concepts
- AWS account setup and navigation
- Core services: EC2, S3, RDS, Lambda basics
- IAM fundamentals and security overview
- Hands-on: Creating first EC2 instance

**Week 2:** [AWS VPC, Networking & Security Architecture](1.2-week2/)
- VPC design and configuration
- Subnets, route tables, and gateways
- Security groups and network ACLs
- Load balancing (ALB, NLB)
- Hands-on: Building secure VPC infrastructure

**Week 3:** [Database Services & Data Management](1.3-week3/)
- RDS (Relational Database Service)
- DynamoDB and NoSQL patterns
- Database backup & recovery
- Data migration strategies
- Hands-on: Multi-tier database setup

**Week 4:** [Storage Solutions & S3 Deep Dive](1.4-week4/)
- S3 storage classes and lifecycle policies
- CloudFront CDN and caching strategies
- S3 security and encryption
- Data transfer & optimization
- Hands-on: Building content distribution network

**Week 5:** [Application Services & Scalability](1.5-week5/)
- Auto Scaling and Elastic Load Balancing
- Container basics (Docker fundamentals)
- Elastic Container Service (ECS) introduction
- Lambda serverless compute
- Hands-on: Creating auto-scaling application

**Week 6:** [Monitoring, Logging & Cost Optimization](1.6-week6/)
- CloudWatch monitoring and dashboards
- CloudTrail audit logging
- Cost optimization strategies
- Reserved instances and savings plans
- Hands-on: Setting up comprehensive monitoring

**Week 7:** [Well-Architected Framework & Best Practices](1.7-week7/)
- Five pillars review (Operational Excellence, Security, Reliability, Performance, Cost)
- Design patterns and best practices
- Disaster recovery and business continuity
- Architecture review workshop
- Hands-on: Architecting production-ready system

### Phase 2: Frontend & Cloud Integration (Weeks 8-11)

**Week 8:** [Frontend Development Setup & Frameworks](1.8-week8/)
- Frontend fundamentals (HTML, CSS, JavaScript)
- React.js introduction and component architecture
- State management basics
- Build tools and deployment
- Hands-on: Creating first React application

**Week 9:** [Cloud-Connected Frontend Development](1.9-week9/)
- AWS Amplify framework
- Frontend authentication with Cognito
- API integration with AWS AppSync
- Real-time data synchronization
- Hands-on: Building cloud-connected React app

**Week 10:** [Full-Stack Integration & CI/CD Pipelines](1.10-week10/)
- GitHub/Git version control
- AWS CodePipeline setup
- Continuous Integration with CodeBuild
- Continuous Deployment strategies
- Hands-on: Implementing complete CI/CD workflow

**Week 11:** [DevOps Practices & Infrastructure Automation](1.11-week11/)
- Infrastructure as Code (IaC) with CloudFormation
- Environment management
- Deployment automation
- Blue/Green and Canary deployments
- Hands-on: Automating infrastructure deployment

### Phase 3: Testing, Documentation & Finalization (Weeks 12-13)

**Week 12:** [Testing Strategies & Quality Assurance](1.12-week12/)
- Unit testing frameworks
- Integration testing
- End-to-end testing (E2E)
- Performance testing and optimization
- Security testing basics
- Hands-on: Comprehensive test suite implementation

**Week 13:** [Documentation, Deployment & Program Completion](1.13-week13/)
- Technical documentation best practices
- API documentation
- Deployment procedures
- Knowledge transfer and handover
- Program review and future roadmap
- Hands-on: Finalizing production deployment
