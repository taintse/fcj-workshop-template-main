## CloudJack-21: Where Games Meet Cloud Learning

### From Idea to Reality

"Initially, I wasn't sure whether to join the **AWS Game Builder Challenge**," JC recalls. "But I had a Python code for a blackjack game, and during an Advanced course at AWS Cloud Institute, I **had an idea** â€“ I could **turn each card into an AWS service**, creating an engaging way to teach cloud concepts."

### Game Mechanics

The game **CloudJack-21** combines **classic blackjack mechanics** with **cloud computing education**. Players build virtual cloud infrastructure using **AWS service cards**, with each card representing real services with authentic characteristics.

**The game has two phases:**

**1. Infrastructure Building:**
- Players aim to **build a cloud architecture** with a **total value under 21**

**2. Cloud Battle:**
- Players take on the role of **"Cloud Robber"** or **"Cybersecurity"**
- Use **challenge cards** or **defense cards**

"I wanted to create something **both fun and educational**," JC says. "The challenge was **balancing gameplay with accurate AWS service simulation**."
