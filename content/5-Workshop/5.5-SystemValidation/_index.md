#### Testing Strategy

We have successfully deployed the backend and seeded the admin user. Now, we need to validate the system functionality.

We will simulate a **Client Application** (like a React Native app) making requests to your Serverless Backend.

**Tools:** You can use **Postman**, **Thunder Client** (VS Code Extension), or simply **cURL** in your terminal.
